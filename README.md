# E-Commerce Analytics System

## Project Overview
The E-Commerce Analytics System is a data-driven project designed to analyze customer behavior, sales performance, and business trends. 

This project performs data cleaning, exploratory data analysis (EDA), RFM-based customer segmentation, Customer Lifetime Value (CLV) calculation, and provides business insights through an interactive dashboard.

---

## Objectives
- Analyze sales trends and revenue patterns
- Identify high-value customers using RFM analysis
- Calculate Customer Lifetime Value (CLV)
- Develop an dashboard for visualization
